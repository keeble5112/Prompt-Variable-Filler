# Prompt 变量填充器

Prompt 变量填充器是一款专为 AI 提示词创作者与高频使用者设计的单文件网页端效率工具。它无需任何复杂的后端环境，直接运行在浏览器中。

通过直观的交互界面，它可以自动解析提示词模板中的变量并生成网格化输入框，同时支持本地文件夹深度读取以及利用大语言模型一键智能提取、分组及分析变量。

## 核心特性

* ⚡ 即时变量检测与动态网格
  * 自动抓取模板中所有的变量。
  * 自动生成自适应高度的输入卡片网格，支持一键移除变量恢复原文。
  * 双栏同步滚动，修改变量实时预览最终结果。
* 📂 本地 Prompt 库深度打通 (FileSystem API)
  * 支持选择本地文件夹，构建高还原度的文件夹目录树。
  * 双重读写机制：支持直接无痕刷新本地改动，支持右键直接永久删除本地物理文件。
* 🤖 AI 智能变量拆解 (LLM 语义提取)
  * 集成 LLM 接口，支持自定义 API Base URL、模型与密钥。
  * 三种提取颗粒度：粗颗粒度（大概念层）、中等颗粒度（均衡模式）、细颗粒度（高精拆解）。
  * 提供强大的智能提取审查弹窗，以分组卡片形式呈现 AI 建议，支持自主勾选或放弃更改后一键应用。
* 🎨 高级交互与无缝剪贴
  * 划词设为变量：在模板中鼠标选中任意词汇，悬浮工具栏一键转为自定义变量。
  * 反向聚焦定位：在最终结果中点击任何高亮变量，页面将平滑滚动并自动聚焦、全选对应的变量控制台输入框。
  * 智能导出：支持保留变量格式导出，文件名自动根据 AI 提取的元数据格式化。

## AI 配置指南

若要启用 AI 提取功能：
1. 点击右上角的 API 配置。
2. 输入您的 API 凭据：
   * Base URL 填写您的接口地址（例如常用的大模型中转地址）
   * Model 填写模型名称（例如 gpt 4o mini 或者 deepseek chat）
   * API Key 填写您的密钥
3. 点击保存（API 凭据将长久保存在您的浏览器本地缓存中，数据安全不上传）。

## 文件夹导入兼容说明

本项目使用现代浏览器的 File System Access API：
* 在 Chrome、Edge 或 Opera 等 Blink 内核浏览器中，您可以直接导入本地文件夹，享受物理级的无痕刷新、无痛读取与右键永久删除功能。
* 在旧版浏览器或不兼容的运行环境中，系统会自动降级为兼容文件树读取模式。
